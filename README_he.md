# 🟩 Matrix Website

אפליקציית רשת אינטראקטיבית ומושקעת ב-React בהשראת סרטי "המטריקס", המשלבת **ממשק משתמש (UI) חי**, **ויזואליה בתלת-ממד (3D)** ו**יצירת מסמכים דינמית**.

פרויקט זה נבנה כאתגר פיתוח צד-לקוח (Frontend) במטרה לחקור תבניות React מודרניות, אנימציות מתקדמות, ממשקי משתמש יצירתיים ושילוב עוצמתי של עולמות התלת-ממד בדפדפן.

### 🌍 **[צפייה בפרויקט החי (Live Demo)](https://ofirpeer07.github.io/matrix-website)**

---

## 🚀 סקירת הפרויקט

ה-Matrix Website היא אפליקציית Frontend בעלת זהות ויזואלית חזקה ומוקפדת בהשראת עולם המטריקס.  
הפרויקט שם דגש על **ארכיטקטורה מבוססת רכיבים (Component-based)**, **תנועה ואפקטים**, ו**פיצ'רים למשתמש הקצה**, ומכניס את המשתמש לסביבת "טרמינל" של האקרים.  
האפליקציה כוללת אזורים אינטראקטיביים כמו **המסלול של ניאו (Neo's Pathway)** (הכולל בונה קורות חיים עוצמתי, מתחם וידאו ומדריכים) ו**מחלקת הסוכן סמית' (Agent Smith Department)** (הכוללת חדשות טכנולוגיה ומדריכי פתרון תקלות).

---

## ✨ פיצ'רים מרכזיים

- 🎬 **נראות ואווירת מטריקס**: גשם קוד דיגיטלי, ממשק בסגנון טרמינל, וטיפוגרפיה ירוקה וזוהרת.
- 🧩 **ארכיטקטורת React מודולרית**: מבנה רכיבים מאורגן ורחב, המפריד בצורה חכמה בין רכיבי ממשק ולייאאוט.
- 🎥 **אנימציות חלקות**: מעברי עמוד אינטראקטיביים ואנימציות זעירות המונעות על ידי **Framer Motion**.
- 🌐 **אובייקטים דינמיים בתלת-ממד**: סביבות ואובייקטים תלת-ממדיים אינטראקטיביים באמצעות **Three.js** (בשילוב `@react-three/fiber` ו-`@react-three/drei`).
- 📄 **בונה קורות חיים (Resume Builder) מקצועי**: כלי אינטראקטיבי וחזק עם עיצוב בסגנון האקרים, המותאם ונעול לפרופורציות של דף A4 גם בתצוגה נגללת וגם בהדפסה.
- 🖨️ **ייצוא ל-PDF באיכות גבוהה**: יצירת מסמכים בצד הלקוח על ידי **html2canvas** ו-**jsPDF**.
- 🌍 **תמיכה דו-לשונית (אנגלית / עברית)**: מעבר חלק בין קריאה באנגלית לעברית (כולל תמיכת RTL במקומות הנדרשים) בעזרת מערכת תרגום מקומית (`LocaleContext`).
- 📱 **עיצוב רספונסיבי ומותאם**: גיליונות עיצוב ייעודיים (`*.mobile.css`) המבטיחים חוויה אופטימלית במחשבים וניידים כאחד.

## 🛠️ טכנולוגיות מרכזיות

- **תשתית הליבה**: React (Create React App)
- **עיצוב וסגנון**: CSS Modules, Tailwind CSS (תשתיות), וגיליונות עיצוב גלובליים
- **אנימציות**: Framer Motion
- **גרפיקה בתלת-ממד**: Three.js, `@react-three/fiber`, `@react-three/drei`
- **אפקטים (Postprocessing)**: `@react-three/postprocessing`
- **ניתוב (Routing)**: React Router DOM (`HashRouter`)
- **יצירת מבנה PDF**: HTML2Canvas & jsPDF
- **אייקונים**: Lucide React
- **פריסה ואחסון (Deployment)**: GitHub Pages

## 📁 מבנה הפרויקט

```text
matrix-website/
├── public/                # קבצים סטטיים ותוצרי בנייה (Build)
├── src/
│   ├── assets/            # תמונות, אייקונים וקבצי מדיה גלובליים
│   ├── components/        # רכיבי UI ואזורי מפתח באפליקציה
│   │   ├── AgentSmith/    # אזור הסוכן סמית' (חדשות, פתרון תקלות)
│   │   ├── App/           # רכיב השורש וראוטינג (ניתוב העמודים)
│   │   ├── MainPage/      # עמוד הנחיתה (בחירת הגלולה האדומה/כחולה)
│   │   └── Neo/           # אזור ניאו (קורות חיים, מאמרים, וידאו)
│   ├── context/           # ניהול מצבי קונטקסט (כגון שפה - LocaleContext)
│   ├── styles/            # סגנונות CSS מאורגנים לפי פיצ'רים
│   ├── index.js           # נקודת הכניסה של האפליקציה
│   └── index.css          # סגנונות גלובליים (Tailwind ומשתני המטריקס)
├── package.json           # תלויות וסקריפטים להתקנה והפעלה
└── README_he.md           # תיעוד הפרויקט
```

## ▶️ תחילת עבודה

### דרישות קדם
- Node.js (מומלצת גרסה 16 ומעלה)
- npm או yarn

### התקנה
יש לשכפל את המאגר ולהתקין את התלויות:
```bash
npm install
```

### הפעלה בסביבת פיתוח מקומית
להפעלת הפרויקט במחשב:
```bash
npm start
```
האפליקציה תהיה זמינה בכתובת: [http://localhost:3000](http://localhost:3000)

### בנייה (Build) וכתובת רשת
לבניית האפליקציה לקראת העלאתה לשרת ייצור (Production):
```bash
npm run build
```
להעלאתה ל-GitHub Pages:
```bash
npm run deploy
```
